Put bundled OpenSteamTool install files here before publishing or zipping a release.

Expected files:
- OpenSteamTool.dll
- dwmapi.dll
- xinput1_4.dll
- opensteamtool.toml

Optional:
- Any .lua file, copied to Steam\config\lua
- appmanifest_*.acf and .manifest files, copied to Steam\steamapps

The floating installer only routes known file names and extensions.
